# Feature: Project Details

## Goal 
implement `index.html` and responsive layout

## Designs 
![target layout](/assets/specs/spec2.png) 
## Work
Projects will be divided using a combination of sections and divs;
* The page will contain a header that can link back to the main homepage of the website
* Below the header there will a title of the projects page with a heading1 tag
* Underneathe the main heading there will be three images of project examples lined up side by side with titles above each image.
* Below each image there will be a paragraph description, describing the key elements of the page 
* buttons will be included below the description to be able to follow the link to where the projects are displayed, using the href a tags
* The layputs will be responsive where the content will adjust according to the screen that they're displayed on, following the mobile first responsive design
* Users can interact with the buttons and links and there will be a color change when they hover over buttons

## Deliverables-  - What does “done” look like? - Be concrete and testable 
* The page is complete when it displayed the correct images and titles, with a responsive layout
* All the links successfully take the user to the corresponding page
* No formatting issues and adjust accoriding to the screen size using media queries
Creative Styling and hover elements have been correctly applied
