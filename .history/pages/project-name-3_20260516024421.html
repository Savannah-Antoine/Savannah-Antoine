Tic-Tac-Toe Game

Project Objectives

The Goal: programming a tic tac toe game with a 3x3 board with validation to verify if the user enters an invalid option

Rules and Design

The game was programmed to make X as the starting player using a combination of methods and if statements

User Interface

The screen is programmed to clear and refresh after every move displaying an updataed game board when a player wins or when the game ends in a draw.

Final Results

Outcome: A functional Tic-Tac-Toe game with a friendly user interface and input validation
